# StopWatch
⏱️ Precision StopwatchA high-accuracy timer with a clean UI.Features: Start, Pause, Reset, and Lap functionality.Tech Highlight: Uses setInterval with time-correction logic to ensure accuracy.
⏱️ Stopwatch Start: Click the Start button to begin the timer.

Lap: While the timer is running, click Lap to record a specific timestamp without stopping the clock.

Pause/Reset: Use Pause to freeze time and Reset to clear all data and return to 00:00:00.
